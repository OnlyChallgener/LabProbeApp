# Labprobe v0.9.15

本次为 v0.9.15 的 One UI 延迟测试强化包，建议覆盖上一版后重新触发 GitHub Actions。

## 重点改动

- Ping 页面改为“延迟测试”，支持 ICMP、TCP Connect、HTTP HEAD、HTTP GET。
- 增加 IPv6 优先、IPv4 优先、仅 IPv6、仅 IPv4、自动策略。
- 增加 DNS 策略：自动 DNS、优先 AAAA、优先 A、系统默认。
- Ping/延迟测试参数框加入科技蓝小图标，整体卡片高度压缩，页面标题缩小一号。
- 图表 X 轴继续使用真实耗时，不再用次数 × 间隔假算时间。
- 图表固定 1 秒聚合显示，但底层每次采集仍实时记录，尖峰用橙色点标注，丢包用红点标注。
- Y 轴最小刻度改为 30ms，并按 60/90/120/150/300/600/1000 自适应放大。
- 图表右上角增加真实采样率，例如“真实 31.7次/s”。
- 延迟曲线右上角历史按钮可打开居中 One UI 弹窗，保存最近 10 次测试汇总，支持折叠展开和空间占用显示。
- 工具页卡片改为整张可点，不再必须点击“点击进入”。
