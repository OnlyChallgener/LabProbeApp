# LabProbe v0.9.16 build87 · 漫游测试 UI 修复

## 本版重点

- 重新整理无线漫游测试页面配置区，参考路由器工具页的分段目标、输入框和双按钮布局。
- 保留原有 Wi-Fi RSSI / BSSID / 网关延迟采集逻辑，不改 Hub、不改 Router Agent。
- 增加目标模式：路由器+外网、仅路由器、仅外网。
- 增加采样间隔、超时、采样模式入口。
- 漫游图表空状态范围更合理，避免 RSSI 图显示 0~3 这种无意义刻度。
- X/Y 轴字号更小、更贴边，颜色改为弱化文字色。

## 注意

当前“路由器+外网”仍优先沿用原有网关延迟链路，外网目标作为配置与后续扩展入口保留。
