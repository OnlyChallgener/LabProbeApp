# LabProbe 更新日志

## v0.10.3 build131
- MQTT 前台实时同步，断线自动重连并由 HTTP 增量同步兜底，每 5 分钟完整校准。
- 首页健康分和评分细则路由器的动态光晕向周围扩散更明显。

## v0.10.0-alpha1 build126
- 全局 UI V2：新的淡蓝背景、雾面卡片、底部导航、统一弹层和输入组件。
- 工具页功能图标改为全新的非 3D 图标体系；设备图标素材、设备识别和设备逻辑完全保留。
- 端口映射编辑页重新排版：短字段并排、长 IPv6 全宽、文字垂直居中、窄屏自动改单列。
- 6→6 设备选择器按模式显示设备 IPv6 与后 64 位；无可用 IPv6 的设备会明确提示并隐藏。
- 定时映射新增可重复有效期字段，规则到期后再次启动按上一次时长重新计时（需 Hub v0.8.3）。
- SSH 保持现有“连接与命令 + 淡色结果卡片”设计，不改成整页黑色终端。
- Ping、漫游、流量等图表采样、坐标、事件和曲线逻辑未修改。
