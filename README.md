更新：
- “路由 WAN”改成“路由 WAN6”。
- 关注终端离线卡去掉“离线信息已保留”，过滤 null / - / 空字段。
- 每日总结概览改为“上线 / 下线 / VPN-STUN / DDNS / 备注”。
- 每日总结终端情况取消省略号，展示累计在线时长、上下线次数、最后 IP、最后信号。
