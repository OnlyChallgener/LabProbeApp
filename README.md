# 极客网探 v0.9.14

One UI 精修版。

## 更新重点

- 顶部图标导航改为白色大圆角胶囊，减少灰底感。
- VPN / STUN 钥匙图标支持隐私模式，隐藏公网 IPv4 / IPv6 / 端口，点击复制仍复制真实地址。
- 首页今日概览只统计当天事件，避免继续显示昨天概览。
- 首页卡片支持长按浮起拖动排序，松手后自动保存顺序。
- 设置页新增隐私模式开关。
- 版本号统一为 v0.9.14。

Hub 继续使用 v0.7.2，不需要更新 Hub。
