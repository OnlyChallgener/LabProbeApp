# LabProbeApp v0.9.1

基于 v0.9.0 累计修复。

## v0.9.1 重点

1. Ping 实际采样仍按用户选择的 30/100/200/500/1000ms 执行。
2. 图表显示改成自适应聚合：30ms 采样约 250ms 聚合，100ms 约 400ms 聚合，200ms 约 500ms 聚合，500ms 以上按 1s 左右聚合。
3. 图表使用平滑曲线，减少 1 秒一个点带来的折线感。
4. 丢包仍用小红点显示，但每个聚合窗口只显示一个小点，不串线。
5. 峰值用橙色小点提示，统计仍基于原始采样数据。
6. UI 仍固定约 1 秒刷新，避免高频采样导致界面闪动或卡顿。
