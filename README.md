# LabProbe / 极客网探 v0.8.6

本版基于 v0.8.4 累计修复，Hub 与锐捷脚本无需变更。

## 包含 v0.8.4 的更新

- Ping 图表：去掉 AUTO/正常、大统计卡、实线 X/Y 坐标轴。
- X/Y 动态刻度：X 轴最多 6 个，Y 轴最多 5 个，避免拥挤。
- 图表保留淡网格线、蓝色折线和底部一行统计信息。
- DNS 查询记录最多三行：时间+域名一行，A 一行，AAAA 一行；长内容支持横向滑动。
- DNS 查询记录去重：相同结果自动丢弃，最多保存 10 条。

## v0.8.6 新增修复

- Ping 参数区的「次数 / 间隔 / 超时」三小框统一高度、宽度、圆角、边框与内边距。
- 三小框高度调整为舒适紧凑值，避免太高，也避免遮挡文字。
- 数字和下拉文字统一左对齐并垂直居中，箭头与文字居中对齐。
- 间隔下拉不再使用压缩过度的默认输入框外观，改成统一的参数框样式。

## 编译

上传到 GitHub 后运行：Actions → Android APK → Run workflow。
