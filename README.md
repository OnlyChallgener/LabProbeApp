# LabProbe / 极客网探 v0.8.1 统一精修版

本版把此前零散讨论合并为一个统一版本：

- 记录卡片强制两行化：第一行设备 + 状态 + 时间，第二行核心详情。
- 苹果式左滑删除：左滑露出右侧红色圆角删除条，点击删除。
- 下拉菜单统一 Material 3 圆角浮层，减少默认灰底和双层卡片感。
- Ping 参数区压缩、图表坐标和 Material 3 渐变图表保留。
- 每日总结保留用户可编辑备注。
- 全局统一无衬线字体族。
- VPN 卡片使用 WireGuard / OpenVPN / EasyTier 全称，没数据隐藏。

上传到 GitHub 后运行 Actions → Android APK → Run workflow。
