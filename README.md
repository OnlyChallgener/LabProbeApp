# LabProbe / 极客网探 v0.8.4 大一统修复版

本版基于 v0.8.3 继续收敛，Hub 与锐捷脚本无需变更：

- Ping 图表按简洁曲线重做：去掉 AUTO/正常、大统计卡、实线 X/Y 坐标轴。
- X/Y 刻度动态生成：X 轴最多 6 个，Y 轴最多 5 个，避免拥挤。
- 图表只保留淡网格线、蓝色折线和底部一行统计信息。
- Ping 参数三小框继续等宽等高，文字保持垂直居中。
- DNS 查询记录严格最多三行：时间+域名一行，A 一行，AAAA 一行；长内容支持横向滑动。
- DNS 查询记录去重逻辑保留，相同结果自动丢弃，最多保存 10 条。

上传到 GitHub 后运行 Actions → Android APK → Run workflow。
