# LabProbe / 极客网探 v0.8.9

本版基于 v0.8.8 累计修复，Hub 与锐捷脚本无需变更。

## v0.8.8 更新

1. Ping 延迟曲线优化：Y 轴数字下移，0 点与 X 轴 0s 分开显示，观感更自然。
2. Ping 图表右上角新增丢包胶囊，使用紫色主题；丢包点在图表中以红点标记。
3. Ping 延迟曲线左上图标改为紫色，整体和丢包提示统一。
4. 参数卡片与通用卡片标题字号、图标尺寸略微收敛，减少卡片压迫感。
5. 关注终端离线设备不再显示“离线时间 + 最后时间”两个时间，改为“离线时间 + 最后信号”。

## 编译

GitHub Actions → Android APK → Run workflow。


## v0.8.9 更新

1. Ping 参数卡小框和按钮轻量收敛：目标框、次数/间隔/超时、开始/停止高度更协调，不遮挡文字。
2. Ping 测试页和延迟曲线标题字号下调，减少大标题压迫感。
3. 丢包胶囊从图表内部移到图表外部右侧，避免遮挡曲线和刻度；丢包红点仍在图表内标记。
4. 关注终端离线设备信息保留到下一次上线：刷新时若离线字段为空，会沿用上一轮缓存的 IP、SSID、频段、速率、最后信号和离线时间。
5. 清理接口返回的 null/- 字符串，避免卡片显示 null。
