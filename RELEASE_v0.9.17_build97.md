# LabProbe v0.9.17 build97 · roaming entry permission hotfix

## 修复

- 修复点击“无线漫游”后显示“测试准备中”随后闪退的问题。
- 移除进入漫游页面时自动拉起系统权限弹窗的逻辑，避免部分国产系统在页面切换/首帧期间崩溃。
- 漫游页面会先稳定渲染 UI，只做权限状态检查。
- 点击“开始测试”时，如果缺少必要权限，会自动请求权限；不再保留单独的权限按钮。
- 未连接 Wi-Fi 时仍会阻止测试并提示“请连接 Wi‑Fi 后再开始漫游测试”。

## 范围

- 只改 APP 漫游入口权限逻辑。
- 不改 Hub / Router Agent / API。
- 不改漫游算法和图表逻辑。
